# Quick Reference Guide for Inspectors

## Starting the System

1. **Double-click** "Vehicle Inspection" shortcut on desktop
2. Wait for application to open (5-10 seconds)

## Running an Inspection

### Step 1: Select Video
- Click **"Browse Video"** button
- Find and select your inspection video file
- File name will appear in green

### Step 2: Start Inspection
- Click **"START INSPECTION"** button
- Video window will open automatically

### Step 3: View Results
- Watch the video window:
  - **GREEN boxes** = **RUNNING LIGHT** (steady, normal)
  - **RED boxes** = **BLINKING LIGHT** (hazard light)
- Top panel shows total counts
- Results also appear in the application window

### Step 4: Close Video
- Press **'q'** key in the video window to close it
- Or click the X button

### Step 5: Next Inspection
- Click **"Browse Video"** again
- Select next video
- Repeat from Step 2

## Common Issues

### "No video selected"
- **Solution**: Click "Browse Video" and select a file first

### "File not found"
- **Solution**: Check that the video file still exists at that location

### No lights detected
- **Solution**: 
  - Check video quality
  - Ensure lights are clearly visible
  - Try a different video angle

### Video won't play
- **Solution**: 
  - Check file format (MP4 or AVI work best)
  - Try a different video file

## Getting Help

If you encounter problems:
1. Check the application window for error messages
2. Note the error message
3. Contact your supervisor with the error details

## Keyboard Shortcuts

- **'q'** - Quit video window
- **'p'** - Pause/Resume video (in video window)
